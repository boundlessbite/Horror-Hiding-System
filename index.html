Overview & Basic Explanation

The Horror Hiding System allows players to hide inside objects such as lockers, closets, or other hiding spots seamlessly.

When the player interacts with a hiding object:

  1. The player is moved to a hidden location inside the object.
  2. The camera transitions to the hiding object's camera.
  3. The player's original location is saved.
  4. When exiting, the player is returned to their saved location.

While hidden, the player can:

  • Look around (if enabled)
  • Close or open the door (if available)
  • Turn the flashlight on or off
  • Perform other interactions depending on enabled variables

All features are controlled through variables, so you can easily customize behavior directly from the Details Panel.

Migrating It To Your Project

1. Migrate the Files

In the Horror Hiding project:

  1. Locate the folder _Horror_hiding
  2. Right-click it and select Migrate
  3. Choose your target project

This will copy all required files into your own game project.

2. Configure Global Variables

Open FL_Common and find the function GlobalVars.

  • Set your Main Character in the character reference.
  • You may need to change the output type to match your character.
  • Assign your Input Mapping Context in the output.

3. Add Required Inputs

Open your Input Mapping Context and add the following Actions:

  • IA_Interact
  • IA_InteractInside
  • IA_Flashlight
Input Action Example Key
IA_Interact E
IA_InteractInside R
IA_Flashlight F

4. Add Components to Your Character

Open your Main Character Blueprint and add the following components:

  • AC_Hiding (Controls the hiding system logic)
  • AC_Prompt (Controls the interaction prompts)

Using the Hiding Actor

You do NOT need to create a child blueprint of the Horror Hiding BP.

  1. Drag the Horror Hiding BP directly into the level.
  2. Adjust the variables from the Details Panel.

Variables Overview

Static Mesh Variables

You can assign the main hiding object mesh directly from the Details Panel. You can also adjust its Transform, Position, Rotation, and Scale.

Second Static Mesh (Door)
If your hiding object has a door:

  1. Assign the door mesh to Second Static Mesh.
  2. Enable the variable: Has Second Static Mesh = True.

This will automatically activate the door interaction system.

Camera Variables

These variables control the behavior of the camera while hiding:

  • Camera Transform: Controls the position and rotation of the hiding camera.
  • Camera FOV: Adjusts the field of view while inside the hiding spot.
  • Look Around: Enable or disable the player's ability to look around while hiding.
  • Flashlight Options: Enable/Disable flashlight, intensity, and target arm length.
  • Camera Transition: Customize camera transitions (Enable fade transition, adjust switching speed).

General Variables (Hold Interaction)

This allows the player to hold the interaction key instead of tapping it. Useful in scenarios like being chased by enemies, where entering a hiding spot should take time.

  • Hold Interaction (Bool): Enables hold-to-interact behavior.
  • Hold Duration: How long the player must hold the interaction key to hide.

Door Variables

These variables control how the door behaves:

  • Rotatable Door: Enables door rotation.
  • Inverse Door Rotation: Reverses the direction the door opens.
  • Door Speed: Controls how fast the door opens or closes.
  • Door Sound: Allows you to assign a sound effect when the door is used.

Inside Interaction

The system allows interactions while the player is hidden. Currently supported:

  • Door Interaction: If a door mesh is present, the player can open or close the door while hiding.

You can also easily extend this system to add your own custom inside-interactions.

Function Library – Global Settings

There is a Function Library used across the system to handle important references.

  • Location: FL_Common
  • Function: GlobalVars

Inside this function, you can change input system references, player character casts, and the cast used to access AC_Hiding.

Summary

To use the system:

  1. Place the Horror Hiding BP in the level.
  2. Assign a Static Mesh.
  3. Adjust the camera settings.
  4. Enable optional features such as Door interaction, Flashlight, Look around, or Hold interaction.
  5. Configure additional behavior through the Details Panel variables.

No additional setup or child blueprints are required!
